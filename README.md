# SaaS Template

A full-stack SaaS template built with Next.js, Node.js, and modern tools.

## Features

- **Frontend**: Next.js with TypeScript, Tailwind CSS, DaisyUI
- **Backend**: Node.js with Express, TypeScript
- **Auth**: NextAuth.js with Google OAuth and Supabase
- **Payments**: Dodo Payments integration
- **Email**: Mailgun for transactional emails
- **Database**: Supabase
- **State Management**: Zustand
- **Data Fetching**: TanStack Query
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **SEO**: Automated sitemap, meta tags, OpenGraph
- **Dark Mode**: Automatic dark mode toggle

## Getting Started

1. Clone the repository
2. Install dependencies for both client and server
3. Set up environment variables
4. Run the development servers

### Client

```bash
cd client
npm install
npm run dev
```

### Server

```bash
cd server
npm install
npm run dev
```

## Environment Variables

Copy the `.env.example` files and fill in your values.

## Deployment

- Frontend: Deploy to Vercel
- Backend: Deploy to your preferred Node.js hosting
- Database: Supabase
